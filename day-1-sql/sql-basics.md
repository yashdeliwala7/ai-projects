## What is SQL
SQL is used to interact with databases.

It allows us to create tables, insert data, and query data.

CREATE TABLE task (
    id INTEGER,
    name TEXT,
    status TEXT
);

INSERT INTO task (id, name, status) VALUES (1, "task1", "pending");
INSERT INTO task (id, name, status) VALUES (2, "task2", "pending");
INSERT INTO task (id, name, status) VALUES (3, "task3", "pending");

SELECT * FROM task;

## SQL vs NoSQL

SQL is a relational database management system.
NoSQL is a non-relational database management system.

SQL is used for structured data.
NoSQL is used for unstructured data.

SQL is used for transactional data.
NoSQL is used for analytical data.


## Database normalization

Database normalization is a process of organizing data in a database in such a way that it reduces data redundancy and improves data integrity.

## Database normalization benefits

Reduces data redundancy

Reduces data storage space
