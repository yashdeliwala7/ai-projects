🔹 What is vector DB?

👉 Write like this:

A vector database stores data as embeddings (numerical vectors) and allows semantic similarity search instead of exact matching.

🔹 Why needed?

Traditional databases use exact matching. Vector DB helps find similar meaning even if words are different.

🔹 Similarity search

It compares vectors using distance (like cosine similarity or L2 distance) to find closest meaning.

🔹 FAISS

FAISS is a local, fast vector database used for similarity search on embeddings.

🔹 Pinecone

Pinecone is a managed cloud vector database used for scalable production systems.

🔹 Weaviate

Weaviate supports schema, metadata filtering, and hybrid search.
