# Python Basics Practice (Day 1)

## Topics Covered
- Functions (custom functions, list handling, string manipulation)
- API calls using requests
- Virtual environment (venv)
- Package management using pip

## Features Implemented
- Greeting function
- Sum calculation
- Even number filter
- String reversal
- Student score calculation
- API integration to fetch users
- Random user selection with error handling

## Error Handling
- Handled API failures using try/except
- Checked response status codes
- Prevented crashes on empty data

## Run Project
pip install -r requirements.txt
python main.py
